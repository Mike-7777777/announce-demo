# announce-demo

This is a demo of announcement publish and notification, which using spring-boot, SSE, and Vue.js.

这是通知发布功能的demo，使用Spring Boot框架,Vue.js和SSE协议。

The notification uses dynamic component loading and supports hyperlinks.

通知使用动态组件加载，支持超链接。

This Demo is modified from https://github.com/leven-space/SpringBootNotification.

本Demo是基于 https://github.com/leven-space/SpringBootNotification 基础上修改而来的。

## Run Test 测试

Please run this code below after having spring-boot environment.

构建sping-boot环境后执行

```
mvn spring-boot:run
```



Then, use browser visit the localhost http://localhost:8080/Announce.html

然后用浏览器访问 http://localhost:8080/Announce.html



You will get a page like this, click the button and run.

你会得到这样的界面，点击按钮即可运行。
